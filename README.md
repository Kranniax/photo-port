# Photo Port

A React-based photo portfolio application that showcases images in a modern, responsive gallery.

## Description

Photo Port is a single-page application built with React, designed to display a collection of photographs. It features a clean interface for browsing images, with options to view details and navigate through the portfolio.

## Features

- Responsive image gallery
- Modal view for enlarged images
- Category-based filtering
- Contact form for inquiries

## Technologies Used

- React
- JavaScript
- CSS
- HTML

## Installation

1. Clone the repository:

    ```
    git clone https://github.com/yourusername/photo-port.git
    ```

2. Navigate to the project directory:

    ```
    cd photo-port
    ```

3. Install dependencies:

    ```
    npm install
    ```

## Usage

To run the application in development mode:

```
npm start
```

Open [Photo Port](http://kranniax.github.io/photo-port) in your browser to view the app.

To build for production:

```
npm run build
```

## Scripts

- `npm start`: Runs the app in development mode.
- `npm test`: Launches the test runner.
- `npm run build`: Builds the app for production.
- `npm run eject`: Ejects from Create React App (irreversible).
